What is the Document Object Model(DOM)?

It represents the structure of a document (such as HTML or XML) as a tree of objects, allowing programs to manipulate the content, structure, and style of a document dynamically.

How do you select an element from the DOM?

Selecting an element from the DOM can be done using various methods in JavaScript. Here are some common approaches:

  1. getElementById
  2. getElementsByClassName
  3. getElementsByTagName
  4. querySelector
  5. querySelectorAll

What is event delegation in the context of the DOM And why is it useful?

Event delegation is a technique in JavaScript that leverages the event bubbling phase to handle events more efficiently. Instead of attaching event listeners to individual elements, you attach a single event listener to a common ancestor (parent) of the elements that you want to monitor.

  1. Event Bubbling: When an event occurs on an element, it first runs the handlers on that element and then propagates up to its parent elements (this is known as bubbling).
  2. Single Listener: By adding one listener to a parent element, you can handle events for all of its children.
